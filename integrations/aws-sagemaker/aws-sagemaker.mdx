### Steps to load your file from AWS Sagemaker.

* Choose the source as `"File"` and choose` "Model repository"`

* Authenticate and connect your AWS Account.

* Enter `Model Name` and paste the `ARN` link from AWS Sagemaker  
   * A Sample of the ARN link is - `arn:aws:sagemaker:us-east-1:123456781234:model-package/PyTorch-model/1`

* Make sure to enter the input and output parameters required for the model.

* Proceed and load the model into Inferless.
